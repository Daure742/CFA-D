😞 PROBLÈMES COURANTS
───────────────────────────────────────────────────────────────────────────────

ÉCRAN BLANC:
  1. Regarde DiagnosticPanel en bas-droit
  2. Si rouge: lire le message d'erreur
  3. Si vert: ouvre F12 console pour chercher erreurs
  → Lire TROUBLESHOOT.md pour plus de détails

ERREUR CORS:
  Solution: backend/.env CLIENT_URL=http://localhost:5173,...
  Puis redémarre le backend
  → Lire TROUBLESHOOT.md - Erreurs CORS

MONGODB NE SE CONNECTE PAS:
  Vérifier MONGO_URI dans backend/.env
  S'assurer IP whitelistée (MongoDB Atlas)
  → Lire TROUBLESHOOT.md - Problèmes Base de Données

PORT DÉJÀ UTILISÉ:
  Changer PORT dans backend/.env
  Actualiser VITE_API_URL dans cfa_digital/.env.local
  → Lire TROUBLESHOOT.md - Problèmes de Variables
